# Competing Consumers pattern with Spring Boot and Hazelcast Demo project

## Tech-stack
- Spring Boot version: 3.1.1-SNAPSHOT
- Hazelcast version: 5.2.1
- Java version: 20 (need `--enable-preview` for Virtual Threads)
- Build tool: Maven
- Docker
